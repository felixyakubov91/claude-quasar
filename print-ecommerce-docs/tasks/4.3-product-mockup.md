# Task 4.3: Product Mockup

## Goal
Show uploaded image on product template preview.

## Instructions

Create `frontend/src/components/upload/ProductMockup.vue`:

```vue
<template>
  <div class="product-mockup">
    <div class="mockup-container" :style="containerStyle">
      <!-- Product Template -->
      <img :src="templateUrl" class="template-image" />
      
      <!-- User Image Overlay -->
      <div class="user-image-overlay" :style="overlayStyle">
        <img :src="userImageUrl" :style="imageStyle" />
      </div>
    </div>
  </div>
</template>

<script setup lang="ts">
import { computed } from 'vue';

const props = defineProps<{
  templateUrl: string;
  userImageUrl: string;
  placement: {
    x: number;      // percentage from left
    y: number;      // percentage from top
    width: number;  // percentage of container
    height: number; // percentage of container
    rotation?: number;
  };
}>();

const containerStyle = computed(() => ({
  position: 'relative',
  width: '100%',
  maxWidth: '500px',
}));

const overlayStyle = computed(() => ({
  position: 'absolute',
  left: `${props.placement.x}%`,
  top: `${props.placement.y}%`,
  width: `${props.placement.width}%`,
  height: `${props.placement.height}%`,
  overflow: 'hidden',
}));

const imageStyle = computed(() => ({
  width: '100%',
  height: '100%',
  objectFit: 'cover',
  transform: props.placement.rotation ? `rotate(${props.placement.rotation}deg)` : undefined,
}));
</script>

<style scoped>
.template-image {
  width: 100%;
  display: block;
}
</style>
```

## Acceptance Criteria
- [ ] User image shows on product template
- [ ] Position and size configurable
- [ ] Updates in real-time during editing
