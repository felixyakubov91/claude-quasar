## Quick Start

### Prerequisites

- Node.js 18+
- Docker & Docker Compose
- PostgreSQL (or use Docker)

### Development Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/your-org/print-ecommerce.git
   cd print-ecommerce
   ```

2. Install dependencies:
   ```bash
   # Frontend
   cd frontend
   npm install

   # Backend
   cd ../backend
   npm install
   ```

3. Set up environment variables:
   ```bash
   cp frontend/.env.example frontend/.env
   cp backend/.env.example backend/.env
   ```

4. Start development servers:
   ```bash
   # Terminal 1 - Backend
   cd backend
   npm run develop

   # Terminal 2 - Frontend
   cd frontend
   npm run dev
   ```

5. Access the applications:
   - Frontend: http://localhost:9000
   - Strapi Admin: http://localhost:1337/admin

# Cloning Guide

This guide explains how to create a new instance of the platform for a different client.

## Quick Clone

Run the clone script:

```bash
./scripts/clone-project.sh new-client-name
```

This will:
1. Copy the project to a new directory
2. Generate new environment files
3. Prompt for branding configuration

## Manual Steps

### 1. Copy Project

```bash
cp -r print-ecommerce new-client-ecommerce
cd new-client-ecommerce
rm -rf .git
git init
```

### 2. Update Branding

Edit `frontend/src/css/quasar.variables.scss`:

```scss
$primary   : #YOUR_PRIMARY_COLOR;
$secondary : #YOUR_SECONDARY_COLOR;
$accent    : #YOUR_ACCENT_COLOR;
```

### 3. Replace Assets

Replace these files:
- `frontend/public/images/logo.svg`
- `frontend/public/images/logo-dark.svg`
- `frontend/public/favicon.ico`
- `frontend/public/icons/*`

### 4. Update Translations

Edit company-specific text in:
- `frontend/src/i18n/en-US/index.ts`
- `frontend/src/i18n/he/index.ts`

Key translations to update:
```typescript
{
  site: {
    name: 'Your Company Name',
    tagline: 'Your tagline',
  },
  contact: {
    phone: '+972-XX-XXX-XXXX',
    email: 'info@yourcompany.com',
    address: 'Your address',
  },
}
```

### 5. Create External Services

1. **Supabase Project**
   - Go to supabase.com
   - Create new project
   - Enable auth providers
   - Copy URL and anon key

2. **Cloudinary Account**
   - Sign up at cloudinary.com
   - Note cloud name, API key, secret
   - Create upload preset

3. **PayPlus Account**
   - Register at payplus.co.il
   - Get API credentials
   - Configure webhook URL

4. **Resend Account**
   - Sign up at resend.com
   - Verify domain
   - Get API key

### 6. Configure Environment

Create `.env` files with new credentials:

```bash
# frontend/.env
VITE_SITE_NAME=Your Company
VITE_SUPABASE_URL=https://xxx.supabase.co
VITE_SUPABASE_ANON_KEY=your-key
VITE_CLOUDINARY_CLOUD_NAME=your-cloud
VITE_GA_MEASUREMENT_ID=G-XXXXXXXX

# backend/.env
DATABASE_URL=postgresql://...
SUPABASE_URL=https://xxx.supabase.co
SUPABASE_SERVICE_KEY=your-service-key
PAYPLUS_API_KEY=your-key
RESEND_API_KEY=your-key
```

### 7. Set Up Database

```bash
cd backend
npm run strapi -- migration:run
npm run strapi -- seed
```

### Create Clone Script
Create `scripts/clone-project.sh`:

```bash
#!/bin/bash

set -e

if [ -z "$1" ]; then
    echo "Usage: ./clone-project.sh <new-project-name>"
    exit 1
fi

NEW_NAME=$1
SOURCE_DIR=$(dirname "$(dirname "$0")")
TARGET_DIR="../$NEW_NAME"

echo "üöÄ Cloning project to $TARGET_DIR..."

# Copy project
cp -r "$SOURCE_DIR" "$TARGET_DIR"
cd "$TARGET_DIR"

# Clean git
rm -rf .git
git init

# Remove node_modules
rm -rf frontend/node_modules backend/node_modules

# Create new env files
cp frontend/.env.example frontend/.env
cp backend/.env.example backend/.env

# Generate secrets
JWT_SECRET=$(openssl rand -base64 32)
ADMIN_JWT_SECRET=$(openssl rand -base64 32)
APP_KEYS=$(openssl rand -base64 32),$(openssl rand -base64 32)

# Update backend .env with generated secrets
sed -i '' "s/JWT_SECRET=.*/JWT_SECRET=$JWT_SECRET/" backend/.env
sed -i '' "s/ADMIN_JWT_SECRET=.*/ADMIN_JWT_SECRET=$ADMIN_JWT_SECRET/" backend/.env

echo ""
echo "‚úÖ Project cloned successfully!"
echo ""
echo "Next steps:"
echo "1. Update frontend/.env with your Supabase and Cloudinary credentials"
echo "2. Update backend/.env with your database and payment credentials"
echo "3. Update branding in frontend/src/css/quasar.variables.scss"
echo "4. Replace logo files in frontend/public/images/"
echo "5. Update translations in frontend/src/i18n/"
echo "6. Run 'npm install' in both frontend and backend"
echo "7. Start development servers"
```

Make script executable:
```bash
chmod +x scripts/clone-project.sh
```
